floor Class

Defined in: scripts/floor.js:1
Module: APP

Floor translates and maps the coordinates of the location information that is included in the RawData. This is achieved based on Offset and Units which are calculated from every Floor Plan image. FloorPlan Image properties are calculated using utils/floor_mapper.html and stored under globals.js
